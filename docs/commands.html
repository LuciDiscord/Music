Music Command Description
clear-queue Clears the queue for music
filter Sets an audio filter when playing a song/video
filter-help Lists all the available sound filters
loop Loops the music queue
np Shows what song is now playing
pause Pauses the current song that is playing
Play a song from title or URL
queue Shows the music queue of all the songs
shuffle Shuffles the queue so that playback order is random
skip skips the current song that is playing
volume Sets the how loud (1 - 100) a song is played
Utility Command Description
help Shows all the available commands
up Shows how long I have been online
avatar Shows another users (Or your own) Display avatar
time Shows you the current time and date for the bot
ping Shows my ping to the websocket
calc Solves a math problem for you
embed Send a message in an embed as me
